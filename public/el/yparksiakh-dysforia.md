# Υπαρξιακή Δυσφορία

Όταν μεγαλώνεις ως το λάθος φύλο, χάνεις πολλές εμπειρίες που θα έπρεπε να σου ήταν διαθέσιμες αν οι υπόλοιποι ήξεραν για 'σένα. Διανυκτερεύσεις, εκδρομές κάμπινγκ, πρόσκοποι, ψώνια, μαζορέτες, και αθλητισμός. Οι εκδηλώσεις μικτών φύλων μπορεί να σου δημιουργήσουν αρκετά διαφορετικά συναισθήματα αναλόγως με το πώς τις αντιμετωπίζεις, π.χ. η συμμετοχή σε σχολικό χορό, η τέλεση κάποιας θρησκευτικής τελετής, ή ακόμα και η απλή πράξη του φλερτ. Αυτή η δυσφορία μπορεί να είναι και βιολογική προέλευσης, όπως το να έχεις θλίψη που δε γέννησες ή δε θήλασες τα παιδιά σου.

Αυτές οι χαμένες ευκαιρίες μπορούν να εκδηλωθούν ως συναισθήματα απώλειας και πόνου. Επιπλέον, οι αναμνήσεις από πράγματα στα οποία *είχατε* πρόσβαση αλλά δε θα έπρεπε να είχατε, καθώς και γεγονότα που απλώς τελέστηκαν με λάθος φύλο, μπορεί επίσης να σας ξυνίζουν, καθώς αυτά μπορεί να προκαλέσουν αμήχανα αισθήματα. Φανταστείτε να πρέπει να είστε γαμπρός στο γάμο σας ενώ ξέρετε ότι θα έπρεπε να είστε νύφη. Να μεγαλώνετε έχοντας ένα όνειρο για τον τέλειο γάμο, αλλά τελικά να παίζετε το λάθος ρόλο σε αυτόν.

Μερικές φορές η υπαρξιακή δυσφορία μπορεί να εκδηλωθεί πλήρως υπαρξιακά, χτυπώντας με όλη τη θλίψη της χαμένης νιότης. Όλα τα ραντεβού, τα πάρτι, η εφηβική ζωή, ή έστω το να μπορούσατε να έχετε κάνει σεξ με τα σωστά σωματικά μέλη όσο ακόμα το σώμα σας ήταν νέο και όσο δεν είχατε άλλες ευθύνες. Είναι χρόνος που δεν μπορεί ποτέ να ανακτηθεί.

Πολλά τρανς άτομα προσπαθούν να ανακτήσουν μερικές από αυτές τις χαμένες εκδηλώσεις, διοργανώνοντας ή πηγαίνοντας σε κουήρ χορούς, οργανώνοντας διανυκτερεύσεις, τελώντας ανανεώσεις γαμηλίων όρκων, και συμμετέχοντας σε κοινές «τελετουργίες εφηβείας», όπως το να τους βοηθάει μια μητρική φιγούρα να ψωνίσουν το πρώτο τους σουτιέν ή να τους μαθαίνει μια πατρική φιγούρα να ξυρίζονται. Εν τέλει, όμως, η υπαρξιακή δυσφορία είναι κάτι που δεν μπορεί να ανακουφιστεί. Μπορείτε να δημιουργήσετε νέες εμπειρίες για να αντικαταστήσετε αυτές που χάσατε, αλλά ποτέ δεν μπορείτε να γυρίσετε πίσω το χρόνο.

Αυτός είναι ένας από τους πολλούς λόγους για τους οποίους η επιβεβαίωση της τρανς νεολαίας είναι τόσο σημαντική. Τα αγόρια θέλουν να κάνουν αγορίστικα πράγματα, τα κορίτσια θέλουν να κάνουν κοριτσίστικα πράγματα, και τα μη-δυαδικά παιδιά θέλουν να κάνουν ό,τι τους φαίνεται σωστό. Όταν χάσουν μια από αυτές τις εμπειρίες, δε θα το ξεχάσουν αυτό ποτέ.
